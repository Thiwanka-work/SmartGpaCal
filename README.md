🚀 Overview

SmartGPA is a modern academic management system designed for university students to track GPA, CGPA, semester progress, and predict future academic performance using smart analytics.

It provides a clean dashboard, visual analytics, and GPA prediction engine in a fully responsive web interface.

✨ Key Features
📊 Academic Dashboard
Live CGPA & GPA statistics
Credit completion tracking
Semester roadmap visualization
Quick semester entry system
📅 Semester Management
Add / edit / delete semesters
Organized academic history view
Instant CGPA updates
📈 Performance Analytics
📉 GPA trend charts (Chart.js)
🏆 Best & weakest semester detection
📊 Credit completion progress graphs
🔮 GPA Prediction Engine
Target CGPA selection (First Class / Upper / Lower)
Future GPA simulation
Required performance calculator
📝 Course GPA Calculator
Course-by-course GPA calculation
Grade + credit input system
Real-time GPA updates
