🎓 SmartGPA
GPA & CGPA Calculator | Academic Management & Prediction System

🌐 Live Demo: https://www.smartgpa.app

SmartGPA is a modern, interactive academic management system designed to help university students track performance, manage semesters, analyze GPA trends, and simulate future academic outcomes.

Built with a clean UI, real-time analytics, and intelligent prediction features, SmartGPA transforms academic tracking into a simple and visual experience.

✨ Key Features
📊 Academic Dashboard
Real-time CGPA & credit statistics
Completed vs remaining credits visualization
Semester overview timeline (roadmap style)
Quick semester entry system
Full semester history table with management options
📅 Semester Management
Add, edit, and remove semesters easily
Grid-based semester visualization
Organized academic progress tracking
📈 Performance Analytics
GPA trend visualization using Chart.js
Best & weakest semester detection
Credit-weighted CGPA calculation
Degree completion progress indicators (charts & rings)
🔮 GPA Prediction Engine
Target classification selection:
First Class (≥ 3.70)
Second Upper (≥ 3.30)
Second Lower (≥ 3.00)
Future GPA simulation based on expected credits
Required performance calculator to reach target CGPA
📝 Course GPA Calculator
Add courses with grades and credit hours
Automatic GPA calculation (4.0 scale system)
Real-time quality point computation
Interactive grade performance visualization
🛠️ Technology Stack
Frontend: HTML5, CSS3, Vanilla JavaScript (ES6+)
UI Styling: CSS Variables, Glassmorphism, Responsive Design
Charts: Chart.js (CDN)
Fonts: Google Fonts (Inter)
Icons: Custom SVG icons
SEO: Meta tags, Open Graph, Schema.org structured data
